# Prerequisites

Before running the application, you have to install these modules.


# Instruction

Run this command to install dependencies.

```sh
yarn install
```

Run this command to start the database.

```sh
yarn devStartDB
```

Run this command to start the application.

```sh
yarn devStartServer
```
